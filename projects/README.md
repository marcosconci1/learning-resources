# Projects

The projects are organized by difficulty level. Each project contains a template file (`project-template.md`) with problem descriptions, step-by-step TODO checklists and helpful tips to guide your implementation. 

## Easy

| Project | Description |
|---------|-------------|
| **Collatz Sequence** | Explore mathematical sequences and implement the famous Collatz conjecture |
| **Comma Code** | Write a function that takes a list and returns a grammatically correct string |
| **Character Picture Grid** | Print images stored as lists of lists of strings |
| **Table Printer** | Display tabular data with right-justified columns |
| **Fantasy Game Inventory** | Create an inventory system using dictionaries |
| **Guess the Number** | Build an interactive number guessing game |

## Medium

| Project | Description |
|---------|-------------|
| **Password Locker** | Store and retrieve passwords using command-line arguments |
| **Adding Bullets to Wiki Markup** | Automatically add bullet points to text from clipboard |
| **Regex Phone and Email Extractor** | Extract phone numbers and emails from text using regex |
| **Mad Libs** | Create a word replacement program using regex |
| **Rename Files** | Rename American-style dates to European-style in filenames |
| **Backup to ZIP** | Create backup ZIP files of a folder |
| **Multiclipboard** | Save and load pieces of text to the clipboard |
| **Excel to CSV Converter** | Convert Excel spreadsheets to CSV format |

## Hard

| Project | Description |
|---------|-------------|
| **Super Stopwatch** | Build a stopwatch program to track lap times |
| **Multithreaded XKCD Downloader** | Download XKCD comics using multithreading |
| **Simple Countdown Program** | Create a countdown timer with sound notification |
| **Sending Member Dues Reminder Emails** | Automate email reminders using Excel data |
| **Auto Unsubscriber** | Automatically unsubscribe from unwanted emails |
| **Adding Logo** | Add watermark logos to images in a folder |
| **Custom Seating Cards** | Generate personalized seating cards using Pillow |
| **Automatic Form Filler** | Fill out web forms automatically using GUI automation |
| **Instant Messenger Bot** | Create a bot to send automated messages |
| **Looking Busy** | Nudge mouse cursor to prevent idle status |

## Source

All projects are from "Automate the Boring Stuff with Python" by Al Sweigart.
