##  Overview

### Foundation
- **Arrays & Hashing**: Core data structure operations and hash table techniques
- **Two Pointers**: Efficient array/string traversal patterns
- **Stack**: LIFO structure for parsing, monotonic problems, and more

### Intermediate Patterns
- **Binary Search**: Divide and conquer search techniques
- **Sliding Window**: Optimize subarray/substring problems
- **Linked List**: Pointer manipulation and list operations

### Advanced Data Structures
- **Trees**: Binary trees, BST, traversals, and tree properties
- **Tries**: Prefix trees for string operations
- **Heap/Priority Queue**: Min/max heap for ordered processing

### Algorithm Techniques
- **Intervals**: Merging, overlapping, and scheduling problems
- **Greedy**: Locally optimal choices leading to global solutions
- **Backtracking**: Recursive exploration with constraint satisfaction

### Dynamic Programming & Optimization
- **1-D DP**: Linear dynamic programming problems
- **2-D DP**: Matrix-based optimization and path problems
- **Graphs**: DFS, BFS, shortest paths, and graph algorithms
- **Advanced Graphs**: Topological sort, union find, minimum spanning tree

### Specialized Topics
- **Bit Manipulation**: Bitwise operations and optimization
- **Math & Geometry**: Mathematical algorithms and geometric problems

## Structure

Each problem directory contains:
- `problem-template.md` - Problem statement, constraints, examples, and hints
- `problem-solution.py` - Optimized solution with time/space complexity analysis
- `notes.md` - Pattern explanations and alternative approaches
